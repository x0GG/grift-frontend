## Getting Started

Instal dependenices

```bash
npm install
#or
yarn
```

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build static

```bash
npm run build
# or
yarn build
```

Static files will be generated in the `out` folder.
